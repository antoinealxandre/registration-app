# Registration App (2D/3D)

Application Python de recalage fluoroscopie/CT.

## Setup

```bash
cd registration_app
pip install -r requirements.txt
python app.py
```

## Structure

- `registration_app/`: application principale
- `script/`: scripts utilitaires
- `data/`: jeux de donnees locaux (non versionnes)
- `outputs/`: resultats generes (non versionnes)
